# TechCorp SaaS - Company Profile

## Company Overview
**TechCorp SaaS** is a B2B software company offering a comprehensive project management and team collaboration platform launched in 2020. We serve 5,000+ businesses globally, from small startups to enterprise clients.

## Product: FlowSync Platform

### Core Features
- **Project Hub**: Centralized project creation, tracking, and management
- **Team Workspaces**: Collaborative spaces with real-time chat and file sharing
- **Time Tracking**: Built-in time tracking with automated reports
- **Integrations**: 50+ native integrations (Slack, GitHub, Google Drive, etc.)
- **Analytics Dashboard**: Customizable dashboards with real-time metrics
- **API Access**: RESTful API with webhooks for custom integrations

### Pricing Tiers
- **Starter**: $29/user/month - Up to 10 users, basic features
- **Professional**: $49/user/month - Unlimited users, advanced features
- **Enterprise**: Custom pricing - All features + dedicated support, SSO, custom contracts

### Technical Stack
- Cloud-based SaaS (AWS)
- Available on web, iOS, Android
- REST API with OAuth 2.0 authentication
- 99.9% uptime SLA
